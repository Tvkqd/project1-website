Prepare to be a Computer Science Major

How to prepare to be a Computer Science Major

In your class, you will probably learn most of the terminology you will need. However, here are some basic terms that will be helpful to know.

program/code/software/application - computer instructions

Programming - writing code to make programs

Programming Language - the type of code used to make the program. Some programming languages are for specific uses while others are very versitile.

Some popular languages:
C, C++, C#
Java
Python
HTML/CSS
JavaScript

Binary - number system with two digits: 0s and 1s

Bit - A binary digit. The smallest piece of information

bool/boolean - a data type that can only be set to true or false

int - integer data type (whole numbers)

float - floating point number (can have decimals)

double - floating point number with more precision than a float

array - sequential set of data

IDE (integrated development environment) - These are software used to make writing and executing code easier.

LAN (local area network) - a network in a local area. An example of this is WiFi

Git - source code control software

Repository - project on source code control such as Git

Push - upload your code to Git

Pull - download updates from Git

Commit - update your copy of code so you can push it



No matter what programming language you want to learn and use, Google will be a great help!
Remember to use key words such as the language you will be using and what you want to do or need help with.


The main resources you should know of are StackOverflow and YouTube.
